`PuGo` 支持多语言环境编译内容。语言文件保存在 `source/lang/language-name.toml(ini)`。

默认的语言文件中有一些必要元素的翻译：

```toml
# file: source/lang/en.toml

# navigation
[nav]
home = "Home"
archive = "Archive"
about = "About"
source = "Source"

# about post
[post]
list = "All Posts"
archive = "Archives"
tag = "Tags"
readmore = "Read More"

# about pagination
[pager]
prev = "Prev"
next = "Next"
```

你可以添加在block中添加自己的内容：

```toml
[block]
item1 = "Item1"
item2 = "Item2"
```

模板文档 [国际化](/zh/docs/tpl/i18n.html) 说明如果使用多语言渲染模板。

### 全局语言设置

在 meta 中 `lang` 可以设置全局默认的语言：

```toml
[meta]
.....

# make sure there is a language file in this the language name
lang = "en"
```

### 单独页面

`Page` 可以读取 front-matter 中的 `lang` 使用特定语言内容渲染：

```toml
# make sure there is a language file in this the language name
lang = "en"
```
